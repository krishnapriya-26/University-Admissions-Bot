University Admissions Bot



Overview

\- AI-powered chatbot designed to assist prospective students with university admission-related queries

\- Provides structured and accurate information about programs, eligibility criteria, application procedures, and deadlines



Objectives

\- Provide instant responses to admission-related queries

\- Reduce manual workload for admission offices

\- Improve accessibility of university information

\- Offer a user-friendly conversational interface



Features

\- Course and program information

\- Eligibility and admission requirements guidance

\- Application process assistance

\- Frequently asked questions handling

\- Interactive chatbot interface



Technology Stack

\- Python

\- Add framework used (Flask / Streamlit / etc.)

\- Git and GitHub



Project Structure

\- main.py – Core application logic

\- requirements.txt – Project dependencies

\- README.md – Project documentation



Installation and Setup

\- Clone the repository

\- Navigate to the project directory

\- Install dependencies using pip install -r requirements.txt

\- Run the application using python main.py



Use Case

\- Students seeking admission information

\- Universities automating inquiry handling

\- Academic support teams improving response efficiency
